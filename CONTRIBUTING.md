# 贡献指南

本仓库主要作为个人项目、毕业设计工程复盘和面试展示材料维护。欢迎通过 Issue 或 PR 补充可复现的问题、文档改进、脱敏数据示例和前端展示修正。

## 提交前检查

- 不提交真实学生信息、图书馆原始数据、数据库 dump 或本地日志。
- 不提交本地环境文件、IDE 配置、密钥、Token、Cookie、`.env` 实例值或运行缓存。
- 文档改动请同时考虑 `README.md` 与 `README.en.md` 的中英文一致性。
- 代码改动请说明影响范围，并尽量附上本地验证命令或截图。

## PR 建议

PR 标题建议使用简短动词开头，例如 `docs: ...`、`fix: ...`、`test: ...`。如果改动涉及功能行为，请在描述中写清楚运行环境、复现步骤和验证结果。
